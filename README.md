# EventosApp

Projeto desenvolvido baseado no projeto disponibilizado pela [Michelli Brito](https://github.com/MichelliBrito) no Youtube.

O projeto é uma aplicação com o intuito de gerenciar eventos e seus convidados.

## Tecnologias

- Maven
- Spring Boot
  - DevTools
  - Spring Web
  - Thymeleaf
    - Materialize Css
  - Spring Data JPA
    - MySQL
